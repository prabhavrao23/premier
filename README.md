# Premier League Stats

Full-stack web app for browsing Premier League player statistics.

Inspired by Erik Kupsa

**Live site:** https://premier-bdftvauj7-prabhav-rao-s-projects.vercel.app

## Stack
- **Backend:** Java 21, Spring Boot 4, Spring Data JPA — deployed on Railway
- **Database:** PostgreSQL on Supabase (587 players)
- **Frontend:** React + Vite — deployed on Vercel

## Features
- Browse all 20 Premier League clubs
- View player cards with goals, assists, xG, xAG, appearances, and minutes
- Filter by name, position, and nation (supports full country names or codes)
